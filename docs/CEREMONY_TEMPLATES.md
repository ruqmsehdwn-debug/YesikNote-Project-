모든 항목은 권장 기본값이며 이동·비활성·추가·복제·삭제·이름 변경이 가능하다. 권장 순서를 벗어나도 경고만 표시한다.

## 화촉점화 Child Item

- 양가 어머님: 입장 → 신랑 어머님 점화 → 신부 어머님 점화 → 맞절 → 내빈 인사 → 자리 이동
- 부모님 A: 신랑 부모님 입장 → 신부 부모님 입장 → 맞절 → 아버님 착석 → 신랑 어머님 점화 → 신부 어머님 점화 → 어머님 내빈 인사 → 자리 이동
- 부모님 B: 신랑 부모님 입장 → 신부 부모님 입장 → 신랑 부모님 점화 → 신부 부모님 점화 → 맞절 → 내빈 인사 → 자리 이동
- 한쪽 혼주님: 입장 → 점화 → 내빈 인사 → 자리 이동
- 생략: 부모 식순을 비활성화하고 현재 입력 보존
- 직접 구성: 기본 Child를 만들지 않고 Owner가 추가
